# Kubernetes Dashboard

透過 Kubernetes Dashboard 提供的簡易的 GUI, 讓使用者能夠簡單的察看Kubernetes Cluster 上的資源分佈與使用狀況

## 部署版本的選擇

-  提供兩種版本
-  部署方式請參考資料夾內的README

|Floder Name | Note |
|:--- | :--- |
|aws-official | AWS 官方版本, 但無法在 Cloud9 演示 |
|unofficial | **可以直接在 Cloud9 演示 (推薦使用)** |
